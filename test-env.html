Environment Variable Test

This page tests if the environment variables are being detected correctly by the LumoraLogin component.

Testing environment variable detection...

Expected Environment Variables:

  • GOOGLE_CLIENT_ID - For legacy Google OAuth mode
  • GOOGLE_CALLBACK_URL - For API Google OAuth mode
  • FRONTEND_URL - For API base URL

How to test:

  1. Make sure your .env file contains the required variables
  2. Restart your development server (npm run dev)
  3. Check the demo page to see if Google Sign-In button appears
  4. Try switching between Legacy and API modes
