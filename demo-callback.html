OAuth Callback Handler

This page demonstrates how the useAuthCallback hook would work in a real application.

This page would normally use the useAuthCallback hook to extract tokens from the URL parameters.

How it works:

  1. User clicks "Sign in with Google" in API mode
  2. Component redirects to GET /auth/google?redirect_uri=this-page-url
  3. Lumora API handles Google OAuth and redirects back here with tokens
  4. useAuthCallback hook extracts accessToken and refreshToken from URL
  5. Tokens are stored in localStorage and user is redirected to dashboard

In your React app, you would use:

import { useAuthCallback } from '@volenday/lumora-login-component'; const AuthCallbackPage = () => { useAuthCallback({ onSuccess: (tokens, user) => { console.log('OAuth successful:', user); window.location.href = '/dashboard'; }, onError: (error) => { console.error('OAuth failed:', error); window.location.href = '/login'; } }); return <div>Processing authentication...</div>; };
